# Praktikum Project14 v1.0.2
Проектная работа по реализации аутентификации и авторизации в проекте Mesto.
## Пакеты, задействованные в проекте:
- Node.js
- Express.js
- Nodemon
- ESLint
- MongoDB
- Mongoose
- bcrypt
- JSON Web Token
и др.
## Как воспользоваться проектом
### Для целей разработки
- Cкачайте архив c копией репозитория [с главной страницы репозитория](https://github.com/1Roderick/praktikumproject14).
- Запустите базу данных командой mongod.
- Установите в проекте необходимые компоненты командой _npm install_.
- Для запуска проекта вызывайте в консоли команду *npm run start* или *npm run dev* (с автообновлением).
